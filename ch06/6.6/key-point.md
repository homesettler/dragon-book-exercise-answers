# 控制流的翻译
## 6.6.1布尔表达式
  B1 || B2 如果B1为真，则不用再计算B2，整个表达式为真
  B1 && B2 如果B1为假，则不用再计算B2，整个表达式为假

## 6.6.2短路代码
  语句
    if(x<100||x>200 && x!=y) x=0;
  代码
      if x < 100 gotoL2
      if False x>200 goto L1
      if False x!=y goto L1
  L2: x = 0
  L1: 接下来的代码
  
  在短路跳转中，布尔运算符&&、||和！被翻译成跳转指令，运算符本身并不出现在代码中，布尔表达式的值是通过代码序列中的位置来表示的。
